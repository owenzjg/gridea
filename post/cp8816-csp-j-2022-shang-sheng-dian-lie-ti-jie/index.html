题目大意

在有 nnn 个点的情况下，补充 kkk 个点
选出最多的点，满足：

  1. 点时相邻的
  2. 坐标单调不递减

骗分技巧

首先观察数据范围，1∼25∼78∼101∼2 5∼7 8∼101∼25∼78∼10 满足 k=0k=0k=0 换成人话就是不需要补充0

在这种情况下非常的简单，按照最长上升子序列（LIS）的做法做就可以了。

设置 f[i] 表示以 iii 结尾的最长上升子序列即可

骗分代码如下

#include <bits/stdc++.h>
using namespace std;
struct node {
    int x, y;
} a[500 + 10];
int f[510];
bool cmp(node a, node b) {
    if (a.x == b.x)
        return a.y < b.y;
    else
        return a.x < b.x;
}
int main() {
    int n, m;
    cin >> n >> m;
    for (int i = 1; i <= n; i++) {
        cin >> a[i].x >> a[i].y;
        f[i] = 1;
    }
    sort(a + 1, a + 1 + n, cmp);
    int maxx = 0;
    for (int i = 1; i <= n; i++) {
        int x = a[i].x;
        int y = a[i].y;
        for (int j = 1; j < i; j++) {
            if (x - a[j].x + y - a[j].y - 1 == 0) {
                if (a[j].y > a[i].y)
                    continue;
                f[i] = max(f[i], f[j] + 1);
                maxx = max(maxx, f[i]);
            }
        }
    }
    cout << maxx;
    return 0;
}

正解

其实将骗分的想明白这道题也就不难了

增加一个维度为 f[i][j] 表示以第 iii 个点结尾，补充 jjj 个点形成的最长上升点列的长度。

边界的条件

f[i][0]=1; //不补充点的时候自己算一个
f[i][j]=j+1;   //补充了j个点，加上自己就是j+1

注：第二个边界其实包括了第一个

第i个点可以由那些点转移过来
假设第i个点可以由点 ttt 转移过来。

  1. t一定要在第i个的点的左下方（包含正左和正下）
  2. 计算第 ttt 个点到第 iii 个点要补充多少个点
    c=a[i].x-a[t].x+a[i].y-a[t].y-1;

状态转移方程
f[i][j]=f[t][j-c]+c+1 (j>=c)

正解代码

#include <bits/stdc++.h>
using namespace std;
struct node {
    int x, y;
} a[500 + 10];
int f[510][110];
bool cmp(node a, node b) {
    if (a.x == b.x)
        return a.y < b.y;
    else
        return a.x < b.x;
}
int main() {
    int n, m;
    cin >> n >> m;
    for (int i = 1; i <= n; i++) {
        cin >> a[i].x >> a[i].y;
        for (int j = 0; j <= m; j++) {
            f[i][j] = j + 1;
        }
    }
    sort(a + 1, a + 1 + n, cmp);
    int maxx = 0;
    for (int i = 1; i <= n; i++) {
        int x = a[i].x;
        int y = a[i].y;
        for (int l = 0; l <= m; l++) {

            for (int j = 1; j < i; j++) {
                if (a[j].y > a[i].y)
                    continue;
                int c = x - a[j].x + y - a[j].y - 1;
                if (l >= c) {
                    f[i][l] = max(f[i][l], f[j][l - c] + c + 1);
                    maxx = max(maxx, f[i][l]);
                }
            }
        }
    }
    cout << maxx;
    return 0;
}

题面：

题目描述

在一个二维平面内，给定 nnn 个整数点 (xi,yi)(x_i, y_i)(xi​,yi​)，此外你还可以自由添加 kkk 个整数点。

你在自由添加 kkk 个点后，还需要从 n+kn + kn+k 个点中选出若干个整数点并组成一个序列，使得序列中任意相邻两点间的欧几里得距离恰好为 111 而且横坐标、纵坐标值均单调不减，即 xi+1−xi=1,yi+1=yix_{i+1} - x_i = 1, y_{i+1} = y_ixi+1​−xi​=1,yi+1​=yi​ 或 yi+1−yi=1,xi+1=xiy_{i+1} - y_i = 1, x_{i+1} = x_iyi+1​−yi​=1,xi+1​=xi​。请给出满足条件的序列的最大长度。

输入格式

第一行两个正整数 n,kn, kn,k 分别表示给定的整点个数、可自由添加的整点个数。

接下来 nnn 行，第 iii 行两个正整数 xi,yix_i, y_ixi​,yi​ 表示给定的第 iii 个点的横纵坐标。

输出格式

输出一个整数表示满足要求的序列的最大长度。
