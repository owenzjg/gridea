题目大意

给出 n,e,d 以及等量关系,求出p，q的值

pq=n;                    （2）
(p-1)(q-1)+1=ed;         （1）

思路分析

首先，看到2式，我们先将其拆开，变为：

pq−(p+q)+2=edpq-(p+q)+2=edpq−(p+q)+2=ed

整理一下，为

p+q=pq+2−edp+q=pq+2-edp+q=pq+2−ed

p+q=n+2−edp+q=n+2-edp+q=n+2−ed

我们看一眼题目数据范围中的第一句话

以下记 m=n−e×d+2m = n - e \times d + 2m=n−e×d+2。

惊奇的发现ccf抄袭了我们的思路。

那我们后面也不妨记 m=n−e×d+2m = n - e \times d + 2m=n−e×d+2。

也就是
p+q=mp+q=mp+q=m

这个时候，我们拥有了p+qp+qp+q和pqpqpq,迅速联想到初一完全平方公式，如下

因为 (p-q)2+4pq=(p+q)2

所以 (p-q)2=m2-4n

所以 p-q=土 sqrt(m2-4n)

我不会告诉你正负号是泥土的土

这个时候我们看一下正负号的必要性。

其实，当取负数或正数的时候，唯一的区别就是 p,qp,qp,q 的位置发生了变化，仅此而已。

所以，我们可以不在乎整个正负号，最终输出的时候按照

cout << min(p, q) << " " << max(p, q) << endl;

就可以了。

我们现在知道了 p+qp+qp+q ans p−qp-qp−q 相减即可。

得到

p=(m+sqrt(m2-4n))/2

q=m-p

由于 p,qp,qp,q 为正整数，所以 (m+sqrt(m2-4n))/2 为正整数。

所以sqrt(m2-4n)为正整数。

至于除以二不用判断，理由如下：

m为奇数时，m2-4n为奇数，开方后仍是奇数，加上m后奇奇得偶，所以不需要判断除以二

m为偶数时，m2-4n为偶数，开放后依然是偶数，偶数加上偶数依然为偶数，所以不需要判断处以二

综上所述，不要判断除以二

代码

#include <bits/stdc++.h>
using namespace std;
#define ll long long
int main() {
    ll k;
    cin >> k;
    while (k--) {
        ll n, d, e;
        cin >> n >> d >> e;
        ll m = n - e * d + 2;
        ll p = sqrt(m * m - 4 * n);
        ll q;
        if (p * p == m * m - 4 * n) {
            p += m;
            p /= 2;
            q = m - p;
            cout << min(p, q) << " " << max(p, q) << endl;
        } else {
            cout << "NO\n";
        }
    }
    return 0;
}
