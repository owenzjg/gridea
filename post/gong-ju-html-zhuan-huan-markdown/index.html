前言

有的时候，在网站看到心意的文章，想要保存下来，以防删除。

有的时候，想把gridea的文章转载到别的地方，却苦于gridea的markdown图片难以处理。

有的时候，我们需要一个好用的工具，丢给他一个链接，它可以还给我们一份markdown。如果你需要，那请往下看。

正文

后期我又找了一个工具，在工具补充（下面），两个哪个顺手用哪个。

这个工具操作方便，通俗易懂。无学习成本，使用价值非常高。（站长我从不打广告，都是肺腑之言）

首先点开链接，你就会发现这个页面怎么这么丑。

但是，外表≠内在，操作一次你就再也脱离不开了。

  1. 首先，按照图中，复制链接到左上角并且按下回车。
  2. 稍等片刻，等待几秒
  3. 右下角复制拿走不谢

还是非常简单的，所以本文就不过多介绍了。

注意

有的时候可能会把你的主题里的一些小附件，比如时间也转换出来，所以不要无脑，还是看一眼欧

工具补充

最近转换微信公众号的文章的时候发现这个工具好像用不了，又发现了一款工具，可以使用这个工具可以获取微信公众号，页面也比较友好，也还是非常好用的,并且是开源的。还有一点就是微信图片有防盗链（就是除了微信其他人都使用不了这个图片），具体解决方案在我的博客-专题-工具里可以查看。
