这道题解法灰常简单，几个字
暴力出奇迹

为什么这么说呢，我这个蒟蒻一人，看到这道题想了半天没有优化的算法于是就产生了dfs的想法，dfs很好写，代码如下：

#include <bits/stdc++.h>
using namespace std;
#define ll long long
ll n;
int t[1000010];
bool check(){
	int a,b,c,d;
	a=0,b=0,c=0,d=0;
	for(int i=1;i<n;i++){
		if(t[i-1]==0){
			if(t[i]==0) b++;
			else if(t[i]==1) a++;
		}else if(t[i-1]==1){
			if(t[i]==0) c++;
			else if(t[i]==1) d++;
		}
	}
	return (a==b && b==c && c==d);
}
void print(){
	for(int i=0;i<n;i++){
		cout<<t[i];
	}
	return ;
}
bool dfs(int step){
	if(step>=n-1){
		if(check()){ print(); return 1;}
		else return 0;
	}
	for(int i=0;i<2;i++){
		t[step]=i;
		if(dfs(step+1)) return 1;
	}
	return 0;
}
int main(){
	scanf("%lld",&n);
	
	if(n==0)printf("-1");
	else if(!dfs(0)) cout<<-1;
	return 0;
}

首先，我们把n转换成行数m

那么我们来逐个分析其与各部分的关系

第二部分：m−1m-1m−1

第三部分：m+1m+1m+1

第四部分：111

这道题数学方法的做法

首先，我们来看下面这幅图找出能提取出来的01和总体长度的关系

首先第一行的关系是

sum=size/2

第二行的关系时

sum=(size-2)/2=size/2-1

相加得

sum=size-1

于是我们就得出来了-1的结论，而非-1得出的结论用“暴力出奇迹”中上述方法找规律即可

题目：

题目背景

「其一」
即使化身白骨，也有不想忘记的东西

已获得转载授权。

题目描述

给定 nnn，请构造一个长度为 nnn 的仅包含 0,10,10,1 的数字串，满足 01,00,10,1101,00,10,1101,00,10,11 出现的次数相等。或报告无解。

这里“出现”指与原字符串中连续的一部分完全相同。例如，在 101110110111011011101 中，01,00,10,1101,00,10,1101,00,10,11 分别出现了 2,0,2,22,0,2,22,0,2,2 次。

请注意，大样例不以文件附加形式给出，而直接放在题目的 输入输出样例 中的 样例 #3

输入格式

一行一个正整数 nnn。

输出格式

若有解，输出一行一个长度为 nnn 的满足条件的仅包含 0,10,10,1 的数字串。

若无解，输出 -1。

样例 #1

样例输入 #1

3

样例输出 #1

-1

样例 #2

样例输入 #2

5

样例输出 #2

00110

样例 #3

样例输入 #3

15400

样例输出 #3

-1
